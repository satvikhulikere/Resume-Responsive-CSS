Redzone Play-Calling Efficiency – NFL

Back to Resume

Description

Analyzed NFL red zone play-by-play and Next Gen Stats data using Python to evaluate the efficiency of different play types, routes, and defensive coverages. Identified trends and insights including:

  • Short routes increased completion rates by approximately 12%
  • Red zone sacks reduced scoring odds by over 20%
  • Runs inside the 5-yard line produced touchdowns on 60% of attempts

Skills Learned

  • Data cleaning and preprocessing of large sports datasets
  • Exploratory Data Analysis (EDA) using Python (Pandas, NumPy)
  • Data visualization for insights using charts and tables
  • Team collaboration on analytics projects
