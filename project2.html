Workout Performance Prediction

Back to Resume

Description

Built regression and time-series models using Python, scikit-learn, and TensorFlow to forecast bench press, squat, and deadlift performance using workout log features. Deployed results in an interactive Streamlit dashboard for user-defined forecasting.

  • Achieved ±2–4% prediction error (RMSE) across lifts
  • Analyzed workout trends and key performance factors
  • Deployed a user-friendly interactive dashboard for predictions

Skills Learned

  • Regression and time-series modeling with Python
  • Data preprocessing and feature engineering
  • Model evaluation using RMSE and other metrics
  • Deployment of ML models in an interactive dashboard (Streamlit)
